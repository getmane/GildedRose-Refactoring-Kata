# Gilded Rose

This is the Gilded Rose kata in Rescript with Jest

## Getting started

Install dependencies

```sh
yarn install
```

## Running tests

To run all tests

```sh
yarn test
```

To run all tests in watch mode

```sh
yarn test:watch
```

To start developing
```sh
yarn dev:rescript
```

To run the TextTest file
```sh
yarn texttest [days]
```
